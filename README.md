A simple and interactive Notes App built using HTML, CSS, Bootstrap, and JavaScript.

It allows users to create, edit, delete, search, and organize notes with categories and a trash system.


Features:
- Create new notes
- Edit existing notes
- Delete notes (move to trash)
- Restore notes from trash
- Search notes in real-time
- Filter notes by category (Personal, Study, Work, Ideas)
- Sort notes (Newest, Oldest, A-Z)
- Pin important notes
- Dark / Light mode toggle
- Local storage support (data saved in browser)


Technologies Used:
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- LocalStorage API

How It Works:

Each note has:
- Title
- Content
- Category
- Date
- Pin status
- Trash status

UI updates dynamically using JavaScript DOM manipulation
